# Employee-Data-Analysis-using-SQL-Tableau
1. Merged multiple tables of Employee datasets and performed Data Anaslysis using MySQL Workbench.
2. Executed joins, Subqueries, and Advanced queries for department-wise analysis of KPIs such as salary, employee counts, gender ratio, and span of control for the company.
3. Transferred the data & organized different charts into an interactive Dashboard using Tableau Public.
4. Tools and technologies used: MySQL, MySQL workbench, and Tableau Public.
